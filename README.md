
# Personalverwaltung

Dieses Projekt ist eine in **C#** entwickelte Konsolenanwendung zur **Personalverwaltung**, erstellt für schulische Zwecke.

## Funktionen
- Hinzufügen von Mitarbeitern
- Anzeigen aller Mitarbeiter
- Filtern nach bestimmten Kriterien
- Speichern und Laden von Daten aus Dateien

## Voraussetzungen
- .NET SDK (Version 6 oder höher)
- Visual Studio oder Visual Studio Code

## Installation und Ausführung
1. Projekt klonen oder herunterladen
2. Im Projektordner folgenden Befehl ausführen:
```bash
dotnet build
dotnet run
